A collection of convenience wrapper classes for creating games in Java.
Requires the JavaMediaUtils package javax.media.utils which can be found at https://github.com/Jonhoo/JavaMediaUtils

All code inspired by Killer Game Programming in Java by Andrew Davison, April 2005, ad@fivedots.coe.psu.ac.th
See http://fivedots.coe.psu.ac.th/~ad/jg/code/index.html and http://fivedots.coe.psu.ac.th/~ad/jg/code/kgpjCode.zip for original source
